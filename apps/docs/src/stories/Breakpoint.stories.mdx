# Breakpoint

Breakpoint is a debugging component shows the currently active Chakra UI breakpoint.

For more information about setting up breakpoints in Chakra UI see their [Default Theme docs](https://chakra-ui.com/docs/styled-system/theme#breakpoints).

## Styling

`Breakpoint` composes Chakra UI [`Box`](https://chakra-ui.com/docs/components/box/usage) so the styles are easily overwritten using the `boxProps` prop.

The default props passed to `Box` are as follows:

```jsx
<Box
  position="fixed"
  left={4}
  bottom={4}
  bg="gray.200"
  px={4}
  py={2}
  rounded="full"
  fontSize="sm"
  fontWeight="semibold"
  zIndex="overlay"
/>
```

To overwrite them, use the `boxProps` prop as follows:

```jsx
<Breakpoint
  boxProps={{
    bg: "red.500",
    color: "white",
  }}
/>
```
